# Code Reviewer Skill

## Overview
This skill enables AI assistants to review code changes, identify issues, suggest improvements, and provide constructive feedback.

## Capabilities
- Analyze code quality and style
- Identify potential bugs and performance issues
- Check for security vulnerabilities
- Suggest refactoring opportunities
- Review test coverage

## Instructions
When asked to review code:
1. Read the entire code change carefully
2. Look for logic errors, edge cases, and potential bugs
3. Check code style and readability
4. Verify error handling is appropriate
5. Suggest specific improvements with reasoning
6. Highlight strengths in the implementation
