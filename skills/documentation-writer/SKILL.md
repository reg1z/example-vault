# Documentation Writer Skill

## Overview
This skill enables AI assistants to generate clear, comprehensive documentation for code, APIs, and projects.

## Capabilities
- Write clear API documentation with examples
- Generate README files and getting started guides
- Document complex algorithms and design patterns
- Create parameter and return value documentation
- Write usage examples and tutorials

## Instructions
When asked to document code:
1. Understand the purpose and functionality first
2. Write clear descriptions using simple language
3. Provide concrete usage examples
4. Document all parameters, return values, and exceptions
5. Include any important notes or caveats
6. Organize documentation hierarchically (overview → details → examples)

## Structure
Use markdown format with:
- H1 for main title
- H2 for sections
- Code blocks for examples
- Lists for parameters/properties
